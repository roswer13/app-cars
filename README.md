# app_cars

This is a simple application for managing a car inventory.
Developed using Flutter and Django.

## Technologies Used

# Frontend
- Flutter
- Dart
- Google Maps API
- Drift
- Bloc
- Secure Storage
- Dio
- GetIt
- Equatable
- GoRouter
- Build Runner

# Backend
- Django
- Django REST Framework
- PostgreSQL
- Docker
- Docker Compose
